# 🐕 Pet Shop

A **full-stack MERN (MongoDB, Express, React, Node.js) Pet Shop web application**.  
This project allows users to browse products, view details, and interact with a dynamic web interface connected to a backend server.


---

## ✨ Key Features
- **Product Listings:** View pets or items with images, names, prices, and categories  
- **React Frontend:** Responsive UI with reusable components  
- **Node.js & Express Backend:** REST API endpoints for products and orders  
- **MongoDB Database:** Persistent storage for product and user data  
- **CRUD Operations:** Add, update, delete, and fetch products (admin-side)  
- **Client-Server Integration:** Frontend communicates with backend APIs dynamically  

---

## 🛠️ Technologies Used
- **Frontend:** React, HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Tools:** npm, Postman (for API testing)
